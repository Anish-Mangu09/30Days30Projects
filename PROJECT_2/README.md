# Day 2: Number Guessing Game 🎯

## Overview

This is Day 2 of my **30 Days 30 Python Projects Challenge**.

The Number Guessing Game generates a random number, and the player must guess it within a limited number of attempts. The program provides hints whether the guessed number is too high or too low.

## Features

* Random number generation
* User input validation
* Higher/Lower hints
* Attempt counter
* Win/Lose messages
* Option to play again

## Technologies Used

* Python 3
* Random Module

## Skills Practiced

* Variables and Data Types
* Conditional Statements
* Loops
* Functions
* Input/Output Handling
* Python Modules

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Anish-Mangu09/30Days30Projects.git
   ```

2. Navigate to the project folder:

   ```bash
   cd PROJECT_2
   ```

3. Run the program:

   ```bash
   python Day02_Number_Guessing_Game.py
   ```

## Learning Outcome

Through this project, I learned how to use Python's random module, validate user input, and build an interactive command-line game.
