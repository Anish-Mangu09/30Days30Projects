# Day 01 - Calculator

## Project Overview

This is a simple menu-driven calculator built using Python as part of the **30 Days 30 Python Projects Challenge**.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Exit Program
* Invalid Choice Validation
* Division by Zero Handling

## Concepts Used

* Variables
* Input/Output
* While Loops
* Conditional Statements
* Arithmetic Operators

## How to Run

```bash
python Day01_Calculator.py
```

## Sample Output

```
--- Menu ---
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
5. Exit Program

Enter your choice (1-5):
```

## Learning Outcome

Learned how to build an interactive console application using loops, conditions, user input, and basic arithmetic operations.
