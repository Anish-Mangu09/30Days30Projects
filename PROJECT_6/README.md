# 🎓 Student Grade Calculator

A simple Python project that calculates a student's total marks, percentage, and grade.

## Features

- Input marks for multiple subjects
- Calculates total marks
- Calculates percentage
- Displays grade
- Validates marks input (0-100)

## Skills Used

- Python Functions
- Loops
- Conditional Statements
- User Input
- Basic Arithmetic

## Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90-100 | A+ |
| 80-89 | A |
| 70-79 | B |
| 60-69 | C |
| 50-59 | D |
| Below 50 | F |

## Run

```bash
python grade_calculator.py
```
